# Mechanical Keyboard Configurations

A collection of custom firmware layouts and flashing instructions for my 65% mechanical keyboards.

---

## TADA68
The classic 65% board. This section contains the original layouts and my personalized modifications.

* **[Setup Guide](tada68/README.md)** – Step-by-step instructions for flashing.
* **[Default Layout](tada68/0.default/README.md)** – The factory standard keymap.
* **[Custom Layout](tada68/1.custom/README.md)** – My optimized layout with modified FN layers.

## XD68 (Xiudi)
A versatile 65% PCB featuring underglow and specific QMK configurations.

* **[Setup Guide](xd68/README.md)** – Step-by-step instructions for flashing.
* **[Default Layout](xd68/0.default/README.md)** – The factory standard keymap.
* **[Custom Layout](xd68/1.custom/README.md)** – My optimized layout with modified FN layers.

## XB67 (KBD67 MKII)

A 65% mechanical keyboard with hot-swap design, USB-C interface, and per-key RGB backlighting.

* **[Setup Guide](xbd67/README.md)** – Step-by-step instructions for VIA and firmware configuration.
* **[Firmware Files](xbd67/firmware/)** – Pre-configured VIA and binary firmware files.
